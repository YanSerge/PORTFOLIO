Task 1:

1. Відкрий браузер (наприклад, Chrome) та DevTools (Ctrl+Shift+I).

2. Введи посилання https://www.makeup.com.ua/ в адресному рядку.

3. Натисни Enter.

4. У DevTools перейди на вкладку Network.

5. Знайди в колонці Name запит "www.makeup.com.ua"

6. Заповни аркуш "Task №1" Google-таблиці для цього запиту.


Task 2:

1. Повтори кроки 1-5 з Task 1 для запиту https://makeup.com.ua/

2. Заповни аркуш "Task №2" Google-таблиці для цього запиту.


Task 3:

1. Не закриваючи DevTools додай будь-який товар до кошика на поточній сторінці сайту.

2. У DevTools проаналізуй запит "cart/", його метод і статус. Для зручності можна скористатися фільтром як зображено на картинці нижче, і прописати слово "cart/" у полі фільтра.

3. Заповни аркуш "Task №3" Google-таблиці для цього запиту.

Task 4:

1. Видали товар з кошика.

2. У DevTools з’явиться ще один запит "cart/". Проаналізуй його метод і статус.

3. Заповни аркуш "Task №4" Google-таблиці для цього запиту.

Task 5:

1. Видали слово “cart/” у полі фільтра (якщо воно введене).

2. Повтори кроки 1-5 з Task 1 для запиту https://www.makeup.ua/

3. Заповни аркуш "Task №5" Google-таблиці для цього запиту.


Task 6:

1. Повтори кроки 1-5 з Task 1 для запиту https://makeup.com.ua/ua/product/00000/

2. У DevTools знайди запит "00000/". Проаналізуй його метод і статус.

3. Заповни аркуш "Task №6" Google-таблиці для цього запиту.


Task 7:

1. Повтори кроки 1-5 з Task 1 для запиту https://graph.makeup.com.ua/chat/user/me

2. У DevTools знайди запит "me". Проаналізуй його метод і статус.

3. Заповни аркуш "Task №7" Google-таблиці для цього запиту.
