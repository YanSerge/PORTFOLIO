Tasks

Створити наступні запити через Postman


Task 1. Через API i Postman створити дошку, в якій ти будеш виконувати всі подальші завдання. Інструкція виконання запиту міститься в документації АПІ Трелло в розділі “Boards”, підрозділ “Create a Board”.


Task 2. Створити колонку на дошці ліворуч - додати ключ pos і його значення top в таблицю “Params”; ключу name в таблиці “Params” надати значення Left. Для цього варто використати ID вашої дошки. Розділ документації ”Lists”, підрозділ “Create a new List”. Додати тести: на перевірку статусу коду; на перевірку часу запиту; на перевірку, що значення “name” це рядок (string).


Task 3. Створити колонку на дошці праворуч - додати ключ pos і його значення bottom в таблицю “Params”; ключу name в таблиці “Params” надати значення Right. Для цього варто використати ID вашої дошки. Розділ документації ”Lists”, підрозділ “Create a new List”. Додати тести: на перевірку статусу коду; на перевірку часу запиту; на перевірку, що значення “name” це рядок (string).


Task 4. Створити картку в лівій колонці, яка була створена в Task 2. Для цього варто використати ID вашої лівої колонки. Розділ документації ”Cards”, підрозділ “Create a new Card”. Додати тести: на перевірку статусу коду; на перевірку часу запиту; на перевірку, що значення “id” це рядок (string).


Task 5. Створити картку в правій колонці, яка була створена в Task 3. Для цього варто використати ID вашої правої колонки. Розділ документації ”Cards”, підрозділ “Create a new Card”. Додати тести: на перевірку статусу коду; на перевірку часу запиту; на перевірку, що значення “id” це рядок (string).


Task 6. Видалити будь-яку картку, яка була створена в Task 4 або Task 5. Для цього варто використати ID картки. Розділ документації “Cards”, підрозділ “Delete a Card”.


Task 7. Отримати інформацію про картки на дошці. Для цього варто використати ID вашої дошки. Розділ документації “Boards”, підрозділ “Get Cards on a Board”.


Task 8. Оновити інформацію про створену дошку - змінити назву дошки та додати опис до неї. Для цього в таблицю “Params” додати два ключі - name та desc із відповідними значеннями. Розділ документації “Boards”, підрозділ “Update a Board”. Додати тести: на перевірку статусу коду; на перевірку часу запиту.
