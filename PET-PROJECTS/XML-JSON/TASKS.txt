Task:

1. Дано список із 4 людей. Кожна людина має унікальний id, ім'я, адресу проживання та контакти.

2. Потрібно описати інформацію про цих людей і занести її до файлів.

3. Ці файли зберегти у форматах JSON та XML.



Покрокова інструкція:

1. Створіть два файли:

    Файл у форматі JSON з інформацією про 4 людей.
    Файл у форматі XML з такою ж інформацією про 4 людей.

2. У файлі JSON:

    Кожен елемент має містити інформацію про людину з унікальним ідентифікатором, ім'ям, адресою та контактною інформацією.

3. У файлі XML:

    Кожен елемент має відповідати вказаній структурі даних, як для JSON.
    Для представлення вкладених елементів використовуйте відповідні теги.

4. Перевірте, щоб файли відображалися коректно у браузері.
